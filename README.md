For testing purpose :
  1. clone or download this repository
  2. cd geocrowd
  3. ionic platform add android
  4. ionic platform add ios (for mac user only)
  5. cordova plugin add https://github.com/Moonware/cordova-cameraserver
  6. ionic run android
